# 论文精读
## 0.摘要
### 讨论的问题：
    带有语义信息的环境点云地图和语义分割后的图像能够很好地完成自动驾驶汽车的视觉定位任务，而无需使用特征描述子。

### 主要贡献：
    1.提出了一种加入语义信息的地图表达方式，比基于特征点的地图占用更少的存储空间；
    2.使用语义地图和语义分割图像，设计了一种新的视觉定位方式；
    3.算法结果与基于传统特征描述子的定位算法相比，在不同季节变化的环境中取得了不错的效果。
